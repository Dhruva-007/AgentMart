# 🤖 AgentMart
### *Decentralized AI Agent Marketplace*

> **Design AI agents. Deploy them. Earn ALGO every time they’re used.**
> A trustless marketplace where AI logic becomes an ownable, monetizable, on-chain asset.

---

## 📌 Overview

**AgentMart** is a decentralized infrastructure platform that connects AI agent creators with users who need intelligent task execution — without intermediaries, without platform lock-in, and without trust assumptions.

Users define AI agents as structured configurations (prompt behavior + output rules), publish them to a live marketplace, and earn **ALGO** automatically every time another user runs a task through their agent.

Payments are enforced by **Algorand smart contracts**. Execution is powered by **external AI APIs**. Identity is your **wallet**.

No platform owns your work. No middleman controls your earnings.

## ❗ The Problem

| Problem | Real-World Impact |
|---|---|
| 🏢 AI tools are centralized | Platforms own your prompts, your workflows, your data |
| 🔒 No ownership of AI logic | You build the intelligence. They capture the value |
| 💸 No creator monetization | Prompt engineers and AI builders earn nothing |
| 🤝 Payments are platform-controlled | Fees, delays, withdrawal limits — all platform decisions |
| 👁️ No execution transparency | Black-box AI, opaque billing, unverifiable outputs |

**The result:** Billions of dollars of AI value is created by users and captured by platforms. Creators have no economic stake. Users have no transparency.

## 🧪 How to Run Locally

### 1. Clone Repository
```
git clone https://github.com/Dhruva-007/AgentMart.git
cd AgentMart
```

### 2. Install Dependencies
```
npm install
cd server
npm install
cd ..
```

### 3. Setup Environment Variables (.env)
```
VITE_APP_ID=YOUR_APP_ID
VITE_RECEIVER_ADDRESS=YOUR_WALLET
VITE_ALGOD_SERVER=https://testnet-api.algonode.cloud
VITE_NETWORK=testnet

APP_ID=YOUR_APP_ID
RECEIVER_ADDRESS=YOUR_WALLET

GROQ_API_KEY=your_key
```

### 4. Start Backend
```
cd server
npm run dev
```

### 5. Start Frontend
```
npm run dev
```

### 6. Open Application
```
http://localhost:5173
```
